# React Job Listing Application

A modern job listing application built with React, Vite, and Tailwind CSS. Browse, add, edit, and delete job postings with a clean and responsive interface.

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- React Router DOM
- JSON Server (Mock API)

## Getting Started

1. **Install dependencies:**

```bash
npm install
```

2. **Start the JSON Server (Backend):**

```bash
npm run server
```

Runs on `http://localhost:8000`

3. **Start the Development Server (Frontend):**
   Open a new terminal and run:

```bash
npm run dev
```

Runs on `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run server` - Start JSON Server
- `npm run lint` - Run ESLint

---

**Note:** Both servers (JSON Server and Vite dev server) need to run simultaneously for the app to work properly.
